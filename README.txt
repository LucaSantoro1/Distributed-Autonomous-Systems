This README is related to the geneneral project folder management.
---------------------------------------------------------------------------------------------------------

The project folders have the following structure:
• task_1 folder --> there are all the python scripts related to the task 1 assignment
• task_2 folder --> there are all the ros2 folders and python scripts related to the task 2 assignment
• report folder --> all the files .tex of our report with all images used (in particular 
                    we have two file.tex, one for each task: DAS_report_template_.tex and
                     Formation_Control.tex) 
• results & videos folder --> there are some simulations of each point of the task 1 and some 
                                videos regarding the task 2

note: to run the task 2, the folder task_2 must be the root folder

Inside each task folder there is another README.txt file, that explains all the operation to run correcly all
scripts.
